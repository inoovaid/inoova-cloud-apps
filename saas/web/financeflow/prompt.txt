Crie um CRM de Serviços Financeiros moderno e completo (nível SaaS), utilizando React + Tailwind no frontend e FastAPI + PostgreSQL no backend, com arquitetura organizada e pronto para produção.

O sistema deve ser focado em gestão financeira de clientes, contas e serviços, com as seguintes funcionalidades:

1. DASHBOARD
- KPIs: receita total, contas pagas, pendentes, vencidas, fluxo de caixa
- Gráficos (entrada vs saída, mensal)
- Resumo financeiro em tempo real

2. FINANCEIRO (PÁGINA PRINCIPAL)
- Filtros no topo:
  - Botão "Adicionar Conta"
  - Período (data inicial e final)
  - Seleção de Plano de Conta
  - Botão "Relatório"
- Cards de resumo:
  - Vencidas
  - Vence Hoje
  - Vence Amanhã
  - Recebidas
  - Total
  - Todas Pendentes
- Lista de registros financeiros:
  - Exibir contas a pagar e receber
  - Colunas: descrição, valor, data, status, cliente
  - Estado vazio: "Nenhum registro encontrado"

3. CONTAS
- Contas a pagar
- Contas a receber
- Parcelamentos
- Recorrências automáticas

4. CLIENTES
- Cadastro completo
- Histórico financeiro por cliente
- Relacionamento com contas

5. COMISSÕES
- Cálculo automático
- Relatórios por usuário

6. AUTOMAÇÕES
- Lembretes de vencimento
- Notificações
- Regras automáticas (ex: atraso → status vencido)

7. USUÁRIOS E PERMISSÕES
- Login com JWT
- Perfis (admin, financeiro, vendedor)
- Controle de acesso por módulo

8. RELATÓRIOS
- Exportação PDF
- Filtros avançados
- Fluxo de caixa detalhado

9. UI/UX
- Design moderno estilo SaaS (Stripe/Notion)
- Sidebar com:
  - Dashboard
  - Financeiro
  - Clientes
  - Relatórios
  - Configurações
- Layout responsivo
- Cards e tabelas bem organizados

10. EXTRAS DIFERENCIAIS
- Integração futura com API bancária
- Sistema de tags
- Busca e filtros rápidos
- Preparado para multiempresa (multi-tenant)

11. ARQUITETURA
- Separação clara frontend/backend
- Estrutura de pastas organizada
- Pronto para Docker
- Código limpo e reutilizável

ENTREGA:
- Código completo frontend + backend
- Models do banco
- Rotas FastAPI
- Componentes React separados
- Dados mock para teste

OBJETIVO:
Criar um CRM financeiro profissional pronto para evoluir como produto SaaS.
